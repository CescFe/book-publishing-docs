# book-publishing-docs

Centralized technical and architectural documentation for the Book Publishing platform.

## Purpose

This repository groups the key documentation artifacts that define architecture, security, and long-term technical decisions across the platform.

  - Main project report: `project_report/001_project_report_publishing_platform_francesc_ferrer_v1.0.1.pdf`

  - Report metadata:

    - Version: 1.0.1
    - Date: January 20th, 2026
- `decisions/adr/`
  - Architecture Decision Records (ADRs) documenting accepted technical decisions.
